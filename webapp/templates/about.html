About This Project

The UKT Prediction System is a research-based machine learning application designed to predict Indonesian public university tuition fees (Uang Kuliah Tunggal or UKT) across 11 different tiers using advanced ensemble methods and uncertainty quantification techniques.

üéØ Project Overview

Indonesian public universities implement a stratified tuition fee system called "Uang Kuliah Tunggal" (UKT), which categorizes students into multiple payment tiers (UKT-1 through UKT-11) based on socioeconomic factors. This system aims to provide equitable access to higher education while maintaining institutional financial sustainability.

Research Purpose: This project conducts a comprehensive comparative study of Random Forest and Gradient-Boosted Trees (XGBoost, CatBoost, LightGBM) for multi-output UKT prediction with quantile regression and conformal prediction.

üî¨ Methodology

Machine Learning Models

  • Random Forest: Ensemble of decision trees using bootstrap aggregation
  • XGBoost: Extreme gradient boosting with regularization
  • CatBoost: Gradient boosting with ordered boosting and categorical feature handling (Best Model)
  • LightGBM: Histogram-based gradient boosting for efficiency

Uncertainty Quantification

  • Quantile Regression: Predicts 10th, 50th, and 90th percentiles
  • Conformal Prediction: Provides distribution-free 90% prediction intervals

üìä Performance Metrics

CatBoost (Best Model) Performance

  • R¬≤ Score: 0.9620 (96.2% variance explained)
  • RMSE: 1.102M IDR
  • MAE: 449.5K IDR
  • MAPE: 7.96%
  • Interval Coverage: 88.91%
  • Within 10% Threshold: 77.76% of predictions
